When learning a new technology, it is a good idea to start with familiar logic for a project and concentrate on implementation. The concept of "Todo List" is very simple, but the complexity of such app can be easily scaled.

I have started this project with a simple functionality: add and delete items. Just need to add editing feature and we have a CRUD here.

I have used:

  • services
  • event emitter
  • routing
  • subject
  • enum

Than I have added local storage and Bootstrap icons
